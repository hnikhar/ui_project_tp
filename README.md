# ui_project_tp

In the project directory, you can run:

### Installation Frontend

 ```powershell
npm install 
```

 ```powershell
npm i --save axios
```

 ```powershell
npm i --save greeting
```

### Run the application

```powershell
npm start
```

## Connect to Backend

Clone the repository 
```powershell
git clone https://github.com/hnikhar/backend-ui-project.git
```

### Installation
Run the following command to install Express, Body-Parser, and CORS:

 ```powershell
npm install express body-parser cors
```

### Initialize

```powershell
npm init
```

### Run Your Server:

```powershell
node server.js
```
